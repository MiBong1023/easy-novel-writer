# 문피아 장기 연재 구조 대응표

> 문피아는 편당결제 + 장기 연재가 기본. 25화(1권) 단위 아크를 기준으로 전체 스토리를 설계한다.
> 200화(8권) ~ 500화(20권)가 일반적인 완결 범위.

---

## 25화 아크 내부 구조 (1권 = 25화)

| 블록 | 화수 | 기능 | 핵심 |
|------|------|------|------|
| 도입 | 1~5화 | 상황 세팅 | 새 아크의 상황·갈등 제시, 독자 유입 포인트 |
| 확장 | 6~10화 | 갈등 심화 | 적대·동료·세계관 확장, 긴장 고조 |
| 전환 | 11~15화 | 반전/위기 | 예상 밖 전개, 판이 뒤집힘 |
| 클라이맥스 | 16~20화 | 메인 사건 | 아크 핵심 사건 폭발, 최대 볼륨 |
| 정리+훅 | 21~25화 | 해소+떡밥 | 갈등 일단 해소 + 다음 권 훅 |

---

## 전체 구조 (500화/20권 기준 3막)

### 1막: 확립 (1~100화 / 1~4권)
- **1권 (1~25화)**: 주인공 + 세계관 + 핵심 갈등 확립. **투베 진입 + 유료전환** 최우선.
- **2권 (26~50화)**: 세력 확장, 첫 번째 대립 구도. 유료 독자 안착 구간.
- **3권 (51~75화)**: 1막 위기 고조. 주인공의 한계 노출.
- **4권 (76~100화)**: 1막 클라이맥스. **선독점 해제(100화) → 타플 유통 시작점.**

### 2막: 확장과 시련 (101~300화 / 5~12권)
- 세계 확장, 새 세력·인물 등장, 주인공 성장의 고비
- 2~3개의 중간 대아크(각 50~75화)로 구성
- 중간 지점(150~200화 부근)에서 전체 서사의 대전환
- **매권 마지막 화의 훅이 결제 유지의 핵심**

### 3막: 수렴과 결말 (301~500화 / 13~20권)
- 복선 회수 가속, 최종 적대 세력 등장
- 마지막 3~4권은 최종 결전 + 에필로그
- 완결 후 **E북(전자책) 출간** 가능 (25화 이상 + 완결 필수)

## 단축형 구조 (200화/8권 기준)

| 막 | 범위 | 핵심 |
|---|---|---|
| 1막 | 1~50화 (1~2권) | 확립 + 유료전환 |
| 2막 | 51~150화 (3~6권) | 확장·시련·대전환 |
| 3막 | 151~200화 (7~8권) | 수렴·결전·완결 |

## 문피아 핵심 마일스톤

| 화수 | 이벤트 | 기획 체크 |
|------|--------|----------|
| ~15화 | 초기 반응 확인 | 투베 진입 가능성 판단 (못 들면 40화까지 재도전, 그래도 안 되면 접기 고려) |
| 25화 | 유료전환 기준선 (1권) | 유료전환 훅 필수. "여기서 끊기면 100원 내고 봐야 하는" 클리프행어 |
| 50화 | 2권 완결 / 유료 안정화 | 유료 독자 유지율 체크. 연독률 감소 시 전략 수정 |
| 75화 | 75,000자 초과 → 일반연재 승급 가능 | 자유연재로 시작한 경우 이 시점 체크 |
| 100화 | 선독점 해제 | 타플랫폼(시리즈/카카오) 유통 개시 가능. 유통 전략 수립. |
| 200화 | 중간 분기점 | 단축형이면 완결 구간. 장기형이면 2막→3막 전환. |

## 문피아 투베 전략 체크리스트 (아크1 기획 시 참조)

1. **1화 제목**: 어그로+장르 키워드. 신인은 미친 듯이 어필해야 한다.
2. **1~5화**: 투베 진입용 조회수 확보 구간. 매화 강한 훅 필수.
3. **연재 시간대**: 경쟁 적은 시간대(오후 1시~3시, 주말 오전 피하기) 고려.
4. **N마크**: 3,000자 이상이어야 'N'(새글) 마크 부여. 5,000자 내외가 표준.
5. **연참 전략**: 투베 30위권 진입 시 연참(하루 2편) 고려. 단, 편당 조회수가 기준이므로 무의미한 연참은 역효과.
