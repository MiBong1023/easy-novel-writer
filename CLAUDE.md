# 문피아 웹소설 기획 스킬

## 스킬 로드

이 프로젝트는 문피아 웹소설 기획 전용 스킬을 사용한다.
대화 시작 시 반드시 `SKILL.md`를 읽고 워크플로우를 따른다.

## 트리거 키워드

다음 표현이 나오면 SKILL.md 워크플로우를 활성화한다:

- "웹소설 쓰고 싶다", "연재 기획", "프롤로그 봐줘"
- "에피소드 가이드", "다음 5화 구조", "아크 짜줘"
- "문피아", "웹소설 세팅", "투베", "유료전환", "편당결제"
- 제목 + 프롤로그 텍스트를 그냥 붙여넣는 경우

## 파일 구조

```
CLAUDE.md                          ← 이 파일 (진입점)
SKILL.md                           ← 메인 워크플로우
references/
  ├── genre-tags.md                ← 문피아 장르 태그 (보조 참조)
  ├── hook-patterns.md             ← 장르별 훅 패턴 (보조 참조)
  ├── character-archetypes.md      ← 캐릭터 아키타입 (보조 참조)
  └── plot-structure.md            ← 500화 구조 대응표 (보조 참조)
```

## 운용 원칙

- references/ 파일은 치트시트 수준으로만 참조. 메인은 Claude 내부 지식.
- 작가가 Phase를 건너뛰면 허용. 워크플로우는 유연하게.
- 본문(소설 원고)은 대신 써주지 않는다. 기획·구조 가이드 전용.
